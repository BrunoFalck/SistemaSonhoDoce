# SistemaSonhoDoce
Scripts e documentação de um sistema de banco de dados MySQL fictício

Este repositório é o resultado de um projeto feito na disciplina de Banco de Dados da Faculdade Projeção para a criação de um banco de dados.

Foi utilizada uma doceria real chamada Sonho D'ocê (@Sonho_d.oce no Instagram) para entendermos as necessidades e como um software poderia ajudar nas operaçoes de uma empresa assim, porém todos os dados no BD são FICTÍCIOS.

O banco foi totalmente criado no MySQL Workbench e modelado por https://github.com/BrunoFalck e https://github.com/NFalcao.
